# examples

このディレクトリには、現在の公開 API を確認するための最小実行例を置きます。

## `basic_usage.py`

以下を一通り確認します。

- 初期局面の生成
- 合法手取得
- 着手
- `pack_board` / `unpack_board`
- `play_random_game`
- `supervised_examples_from_trace`
- `encode_planes` / `encode_flat_features`
- `sample_reachable_positions`

実行:

```bash
uv run python examples/basic_usage.py
```
